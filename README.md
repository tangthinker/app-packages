# app-packages

> [English](./README.en.md)

> 应用分发仓库

本仓库用于托管与分发各平台应用安装包，提供稳定、可追溯的版本下载入口。

---

## 仓库说明

- **用途**：集中存放各平台应用的发布包，便于版本管理与分发
- **命名规范**：建议采用 `{app-name}-{platform}-{version}` 格式
- **版本记录**：每次发布请在 Release 或本 README 中注明变更内容

---

## 许可证

各应用安装包的版权与许可以对应产品为准。
